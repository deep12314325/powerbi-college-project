# Indian Domestic Airlines Analysis – Power BI Project

## Overview
This repository contains a comprehensive Power BI report analyzing the performance, pricing, and route characteristics of major Indian domestic airlines. The project includes a central navigation page and six detailed dashboards, each dedicated to one airline. The analysis covers average flight time, pricing trends, route patterns, and operational metrics across the Indian aviation sector.

The report is designed to help users understand airline behavior, compare performance, and explore route‑level insights through an interactive and visually consistent interface.

---

## Overview Dashboard
The Overview page provides a high‑level summary of the entire dataset before users navigate to airline‑specific dashboards. It includes the following key metrics and visuals:

- **Average Price**  
  Displays the mean ticket price across all flights in the dataset.

- **Minimum Price**  
  Shows the lowest fare recorded, helping identify highly competitive routes.

- **Minimum Duration**  
  Highlights the shortest flight duration among all routes.

- **Total Flight Count**  
  A card visual summarizing the total number of flights analyzed.

Additional visuals may include:
- Price distribution charts  
- Route frequency bar charts  
- Time‑based patterns (hourly or daily trends)  
- Airline‑wise comparison snapshots  

This page serves as the analytical foundation of the report.

---

## Navigation Dashboard
The first page of the report acts as a navigation hub. It contains clickable buttons for each airline:

- IndiGo  
- Air India  
- AirAsia India  
- Go First  
- SpiceJet  
- Vistara  

Selecting any airline name takes the user directly to its dedicated dashboard using Power BI bookmarks and page navigation.

## Dataset Description
The dataset includes the following fields:

- Airline name  
- Source and destination airports  
- Flight duration  
- Ticket price  
- Route distance  
- Date and time of flight  
- Flight frequency  
## How to Use
1. Download the Power BI (.pbix) file from this repository.  
2. Open it in Power BI Desktop.  
3. Use the Navigation Page to select an airline.  
4. Explore the dashboards using filters and interactive visuals.
